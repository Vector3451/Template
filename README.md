## INFORMATION

This is an e-commerce website template for hackathons. It includes a product catalog, shopping cart (Zustand + localStorage), checkout flow, and Stripe-ready API routes. Customize based on the problem statement you receive.

For further information regarding the .agents folder, visit : **https://github.com/vudovn/ag-kit**

## DISCLAIMER

Change these to whatever you are building :

- In **main/docs/ARCHITECTURE.md**, change the overview to what your project is supposed to be.
- In **AGENT.md**, change the **PROJECT NAME**, and the **description.**

## THINGS TO DO

Before coming to the hackathons, do the following things first :

1. Clone this repo to the appropriate place where required. **git clone https://github.com/Vector3451/Template.git**.
2. Run `npm install` in the `main/` directory.
3. Run `npm run dev` to verify everything works.
4. Create your own branch before anything begins, so you'll be able to upload your work onto it without conflicts.
5. Replace mock API data (`src/app/api/products/route.ts`) with your real database.
6. Set up Stripe keys in `.env.local` for real payments.
